# Local-AI-for-coding
SANJAY-SAN'S AI is a modern JavaFX desktop application that runs DeepSeek LLM locally using Ollama, providing a fully offline AI assistant with streaming responses, chat history, and a sleek dark UI.
SANJAY-SAN'S AI is a modern desktop AI assistant built using JavaFX and powered by DeepSeek LLM running locally via Ollama.

This application provides a clean, agent-style interface with streaming responses, persistent chat history, and a fully offline AI experience.
